# Employers-Laravel  
  
# Install  
  
git clone https://github.com/BogdanSG/Employers-Laravel.git  
composer update  
  
# Connection DB Settings  
  
PHP: .env  
  
# DataBase  
  
Create DataBase 'employers'  
php artisan migrate  
php artisan db:seed  
  
# Run  
  
php artisan serve or npm start
